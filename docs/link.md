## `netlify-cli link`

Link a local repo or project folder to an existing site on Netlify

```
USAGE
  $ netlify-cli link

OPTIONS
  --force
  --id=id
  --name=name

EXAMPLES
  $ netlify init --id 123-123-123-123
  $ netlify init --name my-site-name
```
